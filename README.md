# SRGB-LED-Unit

## このリポジトリについて
このリポジトリは、電気工学研究部のスマートイルミネーションシステム用 RGB発光ユニットの設計データ・ファームウェアを管理するためのリポジトリです。
<pre>
./
├── Hardmware
│   ├── Circuit/board 回路・基板設計データ
│   ├── Exterier 筐体設計データ
│   └── Doc ドキュメント
└── Firmware
    ├── Test 各種テストファームウェア
    ├── Standalone スタンドアロン動作用ファームウェア
    └── Wireress ワイヤレス制御用ファームウェア
</pre>

## ファームの書き込み方
1. このレポジトリにあるをダウンロードします。
